# College Magnifier
A data visualization system for choosing the best universities

## Development

  ### Environment Requirement 
  - Python 2.7
  - pip
  
  ### Install **flask**
  
  ```bash
  $ pip install -U flask
  ```
  ### Set up Web Server

  ```bash
  $ python run.py
  ```
