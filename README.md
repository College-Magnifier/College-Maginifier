## Development

  ### Environment Requirement
  - Python 2.7
  - pip

  ### Deployment
  - Heroku

  ### Install Heroku Dev CLI

  ```bash
  $ brew install heroku/brew/heroku
  ```

  ### Install **flask**

  ```bash
  $ pip install flask gunicorn
  ```
  ### Set up Web Server

  ```bash
  $ heroku
  ```
